Velocity deploy quick reference
===============================

This repo deploys the live stack on a GCP VM with Docker Compose.
Do not use Cloud Run for this runtime.

Runtime VM fast path:

  cd /opt/option_trading
  bash ./quick_deploy_gcp.sh

Manual VM commands:

  cd /opt/option_trading
  git fetch origin main
  git checkout main
  git pull --ff-only origin main
  test -f .env.compose || cp .env.compose.example .env.compose
  grep -q '^STRATEGY_ENHANCED_VELOCITY=' .env.compose && sed -i 's/^STRATEGY_ENHANCED_VELOCITY=.*/STRATEGY_ENHANCED_VELOCITY=1/' .env.compose || echo 'STRATEGY_ENHANCED_VELOCITY=1' >> .env.compose
  grep -q '^IMAGE_SOURCE=' .env.compose && sed -i 's/^IMAGE_SOURCE=.*/IMAGE_SOURCE=local_build/' .env.compose || echo 'IMAGE_SOURCE=local_build' >> .env.compose
  python3 -m pytest strategy_app/engines/test_velocity_policies.py -q
  sudo docker compose --env-file .env.compose -f docker-compose.yml build strategy_app dashboard
  sudo docker compose --env-file .env.compose -f docker-compose.yml --profile ui up -d strategy_app dashboard

Verify:

  curl -fsS "http://127.0.0.1:${DASHBOARD_PORT:-8008}/api/health" | python3 -m json.tool
  curl -fsS "http://127.0.0.1:${DASHBOARD_PORT:-8008}/api/trading/velocity-testing/test" -G --data-urlencode "date_from=2026-04-11" --data-urlencode "date_to=2026-04-18" | python3 -m json.tool
