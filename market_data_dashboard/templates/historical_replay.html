How To Use This Page

1. Check Run Scope Use `Pinned Run` and `Latest Run Status` first. `Pinned Run` means you are reviewing a specific replay run, not automatically switching to the newest run.
2. Read The Chart Correctly `Underlying Session Chart` is always one selected session day only. It is not a whole-range chart.
3. Read All Trades `All Trades In Selected Run Range` is the multi-day table for the full replay window, such as `2024-01-02` to `2024-01-31`.
4. Move To Compare Use `Evaluation Compare` when you want the run-level result, equity, deltas, and baseline comparison across the whole range.
If replay is still running, expect partial trades, partial signals, and incomplete chart markers until the run reaches `completed`.

Operator Playbook

Scope First Confirm the active `run_id`, replay status, and selected date range before reading any panel.
Outcome First Read `Range Trades` and `Latest Closed Trade` before using votes to explain why a run behaved a certain way.
Exit Read Interpret `TRAILING_STOP` together with `exit_mechanism`. The mechanism is the real owner of the exit.
Alert Read Treat stale data and monitoring warnings as degraded trust conditions, not automatic model failure.
