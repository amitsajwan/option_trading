```python
    logger.warning("direction_ml: %d NaN > threshold ‚Äî abstaining", nan_count)
    return None
```
**Not yet implemented.** Until fixed, direction model always fires even on bad data.

### FM-5 ‚Äî xgboost version mismatch (silent garbage probs)

**Effect:** Model trained on xgboost X but served on xgboost Y ‚Üí `predict_proba` output format changed between 2.x and 3.x ‚Üí all bars return same constant prob (incident 2026-06-13: every bar returned 0.826 ‚Üí 2 identical trades fired).  
**Code path:** `bundle_inference._check_xgboost_version` ‚Äî logs ERROR but **does not abort inference**.

```bash
# Fix: pin xgboost version in Dockerfile to match training version
```

**Verify versions match:**


### FM-6 ‚Äî Feature medians missing from bundle

**Effect:** NaN features filled with `0.0` instead of training-time median ‚Üí model output shifted toward the 0.0-input region ‚Üí systematic bias, often manifests as prob stuck near a constant value.  
**Code path:** `bundle_inference.py:213`: `medians.get(feature, 0.0)` ‚Äî silent fallback to 0.0.

**Detect:**
```bash
python3 -c "
import joblib
for f in ['/opt/option_trading/models/nifty_entry_bundle.joblib',
          '/opt/option_trading/models/nifty_direction_v1.joblib']:
    b = joblib.load(f)
    n_feats = len(b.get('features', []))
    n_medians = len(b.get('feature_medians', {}))
    missing = n_feats - n_medians
    print(f.split('/')[-1], f'features={n_feats} medians={n_medians} missing={missing}')
"
# Good:  nifty_entry_bundle.joblib features=9 medians=9 missing=0
# Bad:   nifty_direction_v1.joblib features=75 medians=0 missing=75  ‚Üê all NaN ‚Üí 0.0
```
If `missing > 0` on a direction model, the output is unreliable whenever any feature is NaN.

### FM-7 ‚Äî Degenerate model output (all same prob)

**Effect:** Every bar returns the same probability. Threshold may fire on every bar or on none. Hard to detect without a distribution check.  
**Root cause:** xgboost version mismatch (FM-5), or model trained on too little data, or feature row is all-zeros due to FM-6.

**Detect:**
```bash
# Grep last 50 entry diag logs and check prob spread:

  awk -F'entry_prob=' '{print $2}' | awk '{print $1}' | sort | uniq -c | sort -rn | head -10
# Good: multiple distinct values spread across [0.02, 0.15]
# Bad:  all the same value (e.g. all "0.826") ‚Üí FM-5; or all "0.049" ‚Üí threshold misconfigured
```
