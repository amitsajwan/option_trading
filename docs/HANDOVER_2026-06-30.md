## Architecture Quick Reference

```

  ↓

  ↓
snapshot_app(_nifty)    — builds snapshots → Redis pubsub (market:nifty:snapshot:v1)
  ↓
strategy_app_nifty      — evaluates each bar → signals → market:nifty:strategy:signals:v1
  ↓                        direction model: 72/75 features, CE/PE prob
  ↓
execution_app_nifty     — places REAL Dhan orders (STRATEGY_INSTRUMENT=NIFTY, lot=scrip_master)
  ↓
persistence_app_nifty   — writes to strategy_positions_nifty, strategy_decision_traces_nifty
  ↓
dashboard               — port 8008, /api/instruments shows both BN+NIFTY
```
