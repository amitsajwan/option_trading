# Manifest Templates

Use these templates to produce dated manifests during fixed-strike remediation:

- `fixed_strike_signoff_TEMPLATE.json` -> `fixed_strike_signoff_YYYYMMDD.json`
- `fixed_strike_delete_manifest_TEMPLATE.json` -> `fixed_strike_delete_manifest_YYYYMMDD_HHMMSS.json`

Do not delete superseded canonical roots until:

1. technical validation is complete
2. architect/trader/data-scientist sign-off is captured
3. delete manifest is generated and reviewed
